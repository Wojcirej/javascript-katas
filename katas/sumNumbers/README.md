# Sum numbers

### Description
This is solution of challenge 1 prepared by [bstashchuk](https://github.com/bstashchuk)

### Examples
```javascript
sumNumbers(1, 2, 3); // 6
sumNumbers(2, 4, 6, 8); // 20
sumNumbers(1, 2, 3, 4, 5); // 15
sumNumbers(1, 3); // 4
sumNumbers(2, 5, 80, 1, 10, 12); // 110
```
